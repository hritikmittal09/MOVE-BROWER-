
# MOVIE BROWSER
SEARCH DETAILS OF ANY HOLLYWOO MOVIE YOU WANT




## Deployment

To deploy this project run

```bash
  npm START
```


## Tech Stack

**Client:** React, Redux, BOOTSTRAP




## Documentation

- WHEN WEBSITE OPENS CLICK ON SEARCH OPTION
- THEN TYPE KEYWORD OF MOVIE TO GET MOVIE DETAILS
